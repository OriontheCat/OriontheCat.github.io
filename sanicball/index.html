Sanicball

Sanicball is a stupid, silly game that throws away complex gameplay mechanics, character progression and microtransactions in favor of pure fast. It features the all the latest dumb internet memes thrown together in a blender to form a truly europhoic circlejerking experience. This wasn't the original intention but it ended up happening anyway. The game is currently in active development again, and has been made fully open source. See below for the GitHub link. If you like the game, feel free to check out my other work here.
